# SpellDash

SpellDashは、英単語を「思い出して打つ」ことで自然に覚える英単語タイピング学習アプリです。

## コンセプト

勉強している感覚を減らし、タイピングしているだけで英単語が定着する体験を目指します。

## 現在の機能

- 60秒タイピング
- 日本語訳から英単語を入力
- ミス時に正解スペル表示
- アルファベットごとの色彩記憶
- スコア計測
- ミス数計測
- 平均タイプ速度
- ベストスコア保存
- 苦手単語の記録
- localStorageによる学習データ保存

## 技術構成

- HTML
- CSS
- JavaScript
- GitHub
- Vercel

## 開発方針

まずはログインなしで、ブラウザ内保存による学習体験を磨く。
その後、ユーザー登録・クラウド保存・復習アルゴリズムを追加する。
