# SpellDash

**英単語を、思い出して打つ。** — スペルを隠して記憶からタイピングする英単語学習ゲーム。

🌐 本番: https://www.spelldash.net

## コンセプト

「勉強アプリ」ではなく **「気付いたら英単語が覚えられているゲーム」**。
Active Recall（思い出す学習）をタイピングゲームの形にし、継続の仕組み（ストリーク・ミッション・デイリーチャレンジ）で毎日開きたくなる体験を作る。

対象: 中学生・高校生・大学受験生・TOEIC学習者・英語を学び直したい社会人。
英語とタイピングを同時に学べるのが独自の強み。

## 主な機能

### 学習コア
- **Study**: 日本語訳から英単語を記憶入力。分からなければEnterで答え表示 → 数問後に再出題（Recall Loop）
- **New Word Learning Loop**: 新出単語は同日に4回自力正解するまで反復 →「今日定着」
- **簡易SRS**: ノーミス連続正解数に応じて復習間隔が伸びる（1〜30日）。複数日にまたがる10連続でMastered
- **苦手のみ復習モード**: 思い出せなかった単語だけを出題。全部クリアで達成演出
- **出題比率スライダー**: テンポ重視⇔新規学習を10%刻みで調整
- **Challenge**: 60秒で定着度を測るスコアアタック
- **発音**: Web Speech API（US/UK切替、自動/手動/OFF）

### 継続の仕組み
- **ストリーク＋シールド**: 連続日数をホームに常時表示。5日ごとにシールド獲得、休んだ日は自動消費で記録を守る
- **Today's Mission**: 今日やるべきReview/Newを提示、達成でボーナスXP
- **Daily Dash**: 1日1回・全ユーザー共通問題の日替わり60秒チャレンジ（日付シードで決定論生成）
- **デイリーランキング**: Daily DashのTOP5＋自分の順位（表示名のみ公開）
- **結果シェア**: Wordle式テキスト（🟩🟨⬛グリッド）をSNSへ
- **XP・レベル・称号・コンボ・効果音**

### 対戦
- **Battle（CPU Ranked / プレシーズン）**: 60秒、CPUと同条件の別問題。Bronze〜Master、RP制。PassはStudyの復習へ接続

### データ
- **Local First**: プレイ中はlocalStorageのみ。Challenge終了時/10語ごと/離脱時にSupabaseへ差分同期
- **クラウド同期**: Google/メールログインで端末をまたいで学習データ・XP・ストリークが復元
- **学習データページ**: スコア推移グラフ・思い出し成功率・習得の進捗・苦手単語・単語一覧
- **PWA**: ホーム画面に追加可、オフラインでも起動

## 技術構成

- フロント: 素のHTML/CSS/JavaScript（ESモジュール、ビルドなし、依存ライブラリなし）
- 認証/DB: Supabase（Google OAuth＋メールマジックリンク、Postgres＋RLS）
- ホスティング: Vercel（mainへのpushで自動デプロイ）
- 単語データ: 1000語・7カテゴリ（中学/高校/TOEIC/ビジネス/IT/旅行/日常）を `data/` にJSONで保持

## 開発

```bash
# ビルド不要。静的サーバーで開くだけ
python3 -m http.server 8000
# → http://localhost:8000
```

- Supabaseのスキーマは `supabase/schema.sql`（SQL Editorで実行）
- `?t=10` でChallenge/Daily/Battleの制限時間を短縮できる（デバッグ用）
- ドキュメント: `docs/STATUS.md`（現状整理）/ `docs/CHANGELOG.md` / `docs/TODO.md`
