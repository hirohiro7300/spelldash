# CHANGELOG

## 2026-07-05

- ヘッダーのアカウントUIをホバー式に刷新。未ログイン時は小さな「ログイン」ボタンだけ表示し、カーソルを合わせる（かクリック）とメール入力パネルが開く
- ログイン時のメニューもホバーで表示されるように変更（ゲームの邪魔にならない）
- hidden属性がCSSのdisplay指定に打ち消されて要素が常時表示されるバグを修正（reset.cssに[hidden]ルール追加）
- ログイン時、ヘッダー右上を頭文字入りの丸いアバターに切り替え（ログイン状態を明示）
- アバターのクリックでメニュー（メール表示・プロフィール・ログアウト）を開閉。外側クリック・Escで閉じる
- プロフィールページ（profile.html）を追加。ログイン時にメール・登録日・学習サマリーを表示
- ヘッダーのメールアドレスをプロフィールへのリンクに変更
- 学習データページにタイピング分析を追加（平均タップ/秒・ミスタイプ率・最高速度・推定WPM など）
- タイピングのセッションデータをlocalStorageに蓄積（storage.js / game.js）
- Supabaseキー未設定時に分かりやすいメッセージを表示するよう改善（Invalid API key対策）
- 学習データを専用ページ（stats.html）にまとめて集約（概要メトリクス・習得の進捗・苦手単語・単語一覧）
- ヘッダーに「プレイ／学習データ」のナビリンクを追加
- ゲーム画面から学習データ・単語一覧セクションを削除しプレイに集中
- ヘッダーを上部固定のナビバーに刷新（ロゴ＋アカウント）
- アカウント操作をヘッダーに統合（未ログイン：メール＋ログイン／ログイン中：メール表示＋ログアウト）
- ヒーローセクションを独立させ、ログイン状態メッセージ欄を追加
- サイトフッターを新設（ブランド・タグライン・リンク・著作権）
- responsive.cssにモバイル対応を追加
