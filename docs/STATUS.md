## 1. プロダクト概要

- **サービス**: SpellDash（https://www.spelldash.net） — 英単語×タイピングの学習ゲーム。日本語訳を見て英単語を記憶から打つ「思い出して打つ」が中核体験
- **ビジョン**: 『世界一継続される英単語サービス』（効率No.1ではなく継続No.1）
- **ターゲット**: 中高生＋社会人の英語学び直し層（スマホファースト）。「英語とタイピングを同時に鍛えられる」が独自価値
- **公式キャラクター**: はすみ（はちゃん）— 学習パートナー（先生ではない・主役にならない）。仕様は docs/CHARACTER.md が正
- **キャッチコピー第一候補**: 「今日も、はちゃんと少しだけ。」（未採用・ヒーローは現在機能説明文）

## 3. 実装済み機能（本番稼働中、PR#32まで全反映）

### ゲームモード（4つ・これ以上増やさない方針）
- **Study**: スペル非表示→記憶から入力。Recall Loop（思い出せなかった語が3〜5問後に再出題、自力正解まで巡回）、新語学習ループ（1日の新語導入数制御＋復習間隔3/5/8問）、苦手のみ復習トグル、出題比率スライダー（回答済み/未回答）、初回3語は短いeasy語スターター
- **Challenge**: 60秒スコアアタック。コンボ・速度ボーナス・ベスト記録
- **Daily Dash**: 1日1回・日付シードで全ユーザー同一40語・完走でロック。Wordle式シェア（🟩🟨⬛テキスト＋1080×1080結果画像）、デイリーランキング（TOP5＋自分の順位、daily_scoresテーブル）
- **Battle**: CPUランクマッチ（Bronze→Master、RP制）。ランク別難易度ミックス（Bronzeはhardなし）、CPUタイプ4種、Passした単語はStudyの復習に接続

### 継続・成長システム
- ストリーク＋シールド（5日ごとに1枚獲得/最大2枚/休んだ日に自動消費）、ヘッダー🔥常時表示（全ページ）
- XP・レベル・称号（Lv100まで）、レベルカードに「あと約N問正解でLv.X」
- Today's Mission（Review/New）、週間サマリー（先週比）、スコア推移グラフ、思い出し成功率
- Knowledge Map Phase A: 語根ファミリー（37family・82語アノテーション）「同じ仲間」カード表示

### はちゃん Phase1（登場は3場面のみ）
- ホーム: 時間帯・状態で変わる一言（プレイ済み＋思い出した語がある日は「今日は◯語思い出せたよ！えらい！」）
- 結果画面: 成長を一緒に喜ぶ一言（ベスト更新は特別セリフ）
- ストリークカード: 記録更新が近い時だけ「あと1日で記録更新！」
- アセット: キャラシートから切り出した96pxアバター2種（通常/お祝い、各23KB）

### スマホUX（主戦場）
- モバイル1行ヘッダー（52px、極小画面はnav横スクロール）
- フォーカスモード: Challenge/Dailyプレイ中は周辺UI全部畳んで「残り時間・単語・入力欄」のみ（1画面1目的）
- モード選択→ゲームカード自動スクロール、完走→リザルト自動スクロール
- ソフトキーボード完全対応（inputイベント照合・予測変換巻き戻し・大文字化対策・enterkeyhint）
- PWA（manifest＋SW Network First＋アイコン）、大型残り時間タイマー（残10秒で赤）

### その他
- オンボーディング「30秒で分かるSpellDash」、効果音7種（WebAudio合成・OFF可）、OGP全ページ、ランキング表示名編集、プライバシーポリシー

## 4. 技術構成

- **フロント**:素のHTML/CSS/JS ESモジュール、ビルドなし。ページ: index/battle/stats/profile/privacy
- **バックエンド**: Supabase（Auth: メールリンク＋Google / Postgres＋RLS）。テーブル: profiles, word_progress, daily_scores, activity_days（すべてRLS適用・SQL適用済み）
- **データ設計**: Local First（localStorageが正、ログイン時に差分同期）。未ログインでも全機能プレイ可（ランキング投稿のみ要ログイン）
- **単語データ**: 1000語・7カテゴリ（中学英語160/高校英語160/TOEIC150/ビジネス140/IT140/旅行130/日常英会話120）、easy/normal/hard 3難易度、ユニークID910
- **難易度ゲート**: Lv1-4=easyのみ→Lv5-9=+normal→Lv10+=全部（Daily除外・プレイ済み語は常に出題可・カテゴリにeasy 0語なら最易レベルへ救済）
- **ホスティング**: Vercel（main pushで自動デプロイ）。CSP等セキュリティヘッダー設定済み・監査済み（docs/SECURITY.md）
- **CDN障害耐性**: esm.sh停止時もSupabase機能だけ静かに縮退しゲームは動く（authProxy/lazyBuilder）
