# SpellDash 設計書

## 目的

SpellDashは、英単語を「見る」のではなく「思い出して打つ」ことで記憶に定着させる学習アプリ。

## 設計思想

- 勉強感を減らす
- 操作はシンプルにする
- 視線移動を減らす
- ミスを学習データとして活用する
- 色・文字・位置・タイピングを組み合わせて記憶させる

## 記憶要素

### 意味記憶

日本語訳から英単語を思い出す。

### 運動記憶

英単語を指で打つ。

### 色彩記憶

アルファベットごとに固定色を持たせる。

### 反復記憶

ミスした単語を優先的に再出題する。

## データ保存

現時点ではlocalStorageを使用。

保存内容：

- playCount
- correctCount
- missCount
- cleanCorrectStreak
- mastered
- masteredAt
