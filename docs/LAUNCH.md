# SpellDash ローンチ＆SNS拡散プラン（Phase 3）

2026-07-15 版。機能は揃った（Daily Dash・ランキング・シェア・OGP・PWA）。あとは点火。

## 拡散の仕組み（実装済み）

1. Daily Dash（全員同じ問題・1日1回）→ 2. 結果シェア（Wordle式テキスト＋画像）→ 3. OGPリッチカードで流入 → 4. オンボーディングで30秒成功体験 → 1に戻る

## 投稿運用

- **毎日**: 自分のDaily Dash結果をシェア（開発者自らループを回す。グリッド絵文字はタイムラインで目立つ）
- **週1**: 開発進捗（スクショ付き）。個人開発の進捗報告はXで拡散されやすい
- **機能ローンチ時**: 案A形式で機能紹介
- ハッシュタグ候補: #英単語 #TOEIC #個人開発 #タイピング #受験勉強

# 小規模ローンチ・キット（PMF探索フェーズ版、2026-07-17追加）

大規模拡散の前に、**5〜10人に直接触ってもらい定性フィードバックを集める**工程。
最重要指標: ①自分が毎日開くか ②兄貴が翌日戻るか ③初回ユーザーがどこで迷うか（docs/PMF.md）

## 配る相手（推奨順）

1. **兄貴**（翌日戻るかの最重要サンプル。翌日「今日もやった？」だけ聞く）
2. 友人・知人 2〜3人（英語学習に興味がある人＋ない人を混ぜる）
3. Xのフォロワー（リプライをくれた人にDMで個別に）

## 声かけテンプレ（LINE/DM用）

> 英単語のタイピングゲーム作ったんだけど、1回だけ遊んでみてくれない？
> スマホで開くだけ、登録不要、60秒で終わる → https://www.spelldash.net
> 遊んだあと「どこで迷ったか」だけ教えてほしい！

※「感想ちょうだい」より「どこで迷ったか教えて」の方が具体的な答えが返ってくる。

## 観察ポイント（横で見られる場合）

- [ ] 開いて5秒以内に「タイプすればいい」と分かったか
- [ ] 最初のEnter（開始）に自力で気づいたか
- [ ] 分からない単語でEnter＝答え表示に気づいたか
- [ ] 完走後に「もう一回」or「Daily」に進んだか、閉じたか
- [ ] はちゃんに反応したか（言及・笑い・無反応）

## 遊んだ後に聞く質問（3つだけ）

1. どこで「ん？」と手が止まった？（迷いポイント）
2. 明日もう一回やるとしたら、何がきっかけになりそう？（継続動機）
3. 誰かに勧めるとしたら誰？（ターゲット検証）

## 記録方法

もらった回答・観察メモは**そのまま開発AIとのチャットに貼る**。
→ AIが迷いポイントを集計し、最大課題1つ→最小実装案（7項目フォーマット）を出す。
